
需要动态链接一些rk的库：
```cmake
set(MPI_LIBS 
    sample_comm
    rockit
    rockchip_mpp
    rkaiq
    rga
    rtsp
    pthread
    dl
)
```
可以从`third_party/luckfox_pico_rkmpi_example/lib/uclibc`中获取

```bash
❯ ls
lib        librkaiq.a    librknnmrt.so       librockchip_mpp.so.0  librockit_full.a   librockit_tiny.a   librockiva.so
librga.a   librkaiq.so   librockchip_mpp.a   librockchip_mpp.so.1  librockit_full.so  librockit_tiny.so  librtsp.a
librga.so  librknnmrt.a  librockchip_mpp.so  librockit.a           librockit.so       librockiva.a       libsample_comm.a
```

使用这个脚本，把这里的`.so`都传送到开发板的 目录中

```bash
# 将所有 .so 文件复制到开发板系统库目录
scp third_party/luckfox_pico_rkmpi_example/lib/uclibc/*.so root@<开发板IP>:/usr/lib/

scp third_party/luckfox_pico_rkmpi_example/lib/uclibc/*.so.1 root@<开发板IP>:/usr/lib/
```

保证可执行文件加载动态库正常
